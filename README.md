# Portfolio

A modern, responsive portfolio website built to showcase professional experience, personal projects, and technical skills. This project features a clean aesthetic with custom typography and interactive components.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons (Feather, Font Awesome, Simple Icons)
- **Typography**: Lexend and Hanken Grotesk
- **Theme**: Custom Context-based Light/Dark Mode

## Features

- **Responsive Design**: Optimized for various screen sizes from mobile to desktop.
- **Dynamic Theming**: Integrated light and dark mode toggle with persistent state.
- **Navigation**: Clean, sticky navigation bar with active section tracking and smooth transitions.
- **Projects Showcase**: Detailed list of projects with descriptions, technology tags, and external links.
- **Experience Timeline**: Structured view of professional work history and achievements.
- **Contact System**: Dedicated contact page with a clean form layout for professional inquiries.
- **Resume Integration**: Direct link to view or download a professional resume in PDF format.
- **Interactivities**: Subtle hover effects, smooth scroll transitions, and micro-animations for an enhanced user experience.

## Project Structure

The project is organized into a clean folder structure:

- `/my-app/src/app`: Contains Next.js routes and page layouts.
- `/my-app/src/components`: Reusable UI components and section layouts.
- `/my-app/src/contexts`: React Context providers for global state management (e.g., Theme).
- `/my-app/src/data`: Static data files for projects, experience, and social links.
- `/my-app/public`: Static assets including images, fonts, and the resume PDF.

## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```

2. Navigate to the project directory:
   ```bash
   cd Portofolio/my-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
